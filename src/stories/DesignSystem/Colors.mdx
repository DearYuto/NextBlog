# Main Color

프로젝트 메인 컬러입니다.

`강조`할 부분에 사용할 수 있습니다.

현재 선택된 요소, 강조할 버튼 등에 사용할 수 있습니다.

<ColorPalette>
  <ColorItem
    title='REASON PRIMARY 🐈'
    subtitle='프로젝트 메인 컬러'
    colors={{ 'Reason Primary': '#7885EC', 'Reason Primary light': '#AEB7FF' }}
  />
</ColorPalette>

# Danger

Danger 컬러는 `삭제`와 같이 중요한 곳에만 사용합니다.

페이지 내에 1회 이상 사용할 수 없습니다.

컬러는 프로젝트 진행 중에 변경될 수 있습니다.

<ColorPalette>
  <ColorItem
    title='Danger'
    colors={{ 'Reason Danger': '#CB4C54', 'Reason Danger Light': '#F57E85' }}
  />
</ColorPalette>

# ETC.

배경 그라데이션 컬러로 사용되는 기타 컬러입니다.

해당 팔레트는 컴포넌트에서 사용할 수 없습니다.

컴포넌트에서 사용할 컬러를 추가해야 한다면 별도의 팔레트를 생성하세요.
