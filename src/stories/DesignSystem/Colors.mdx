## 🎨 컬러 팔레트

프로젝트에 사용할 컬러 팔레트입니다.

# Gray Scale

기본 텍스트 컬러는 `Reason Gray 700`을 사용해주세요.

<ColorPalette>
  <ColorItem
    title='REASON GRAYSCALE 🐈'
    subtitle='그레이 스케일'
    colors={{
      'Reason White': palette.white,
      'Reason Gray 50': palette.gray50,
      'Reason Gray 100': palette.gray100,
      'Reason Gray 150': palette.gray150,
      'Reason Gray 200': palette.gray200,
      'Reason Gray 300': palette.gray300,
      'Reason Gray 400': palette.gray400,
      'Reason Gray 500': palette.gray500,
      'Reason Gray 600': palette.gray600,
      'Reason Gray 700': palette.gray700,
      'Reason Gray 800': palette.gray800,
    }}
  />
</ColorPalette>

# Main Color

프로젝트 메인 컬러입니다.

`강조`할 부분에 사용할 수 있습니다.

현재 선택된 요소, 강조할 버튼 등에 사용할 수 있습니다.

<ColorPalette>
  <ColorItem
    title='REASON PRIMARY 🐈'
    subtitle='프로젝트 메인 컬러'
    colors={{
      'Reason Primary': palette.primary,
      'Reason Primary light': palette.primaryLight,
    }}
  />
</ColorPalette>

# Danger

Danger 컬러는 `삭제`와 같이 중요한 곳에만 사용합니다.

페이지 내에 1회 이상 사용할 수 없습니다.

컬러는 프로젝트 진행 중에 변경될 수 있습니다.

<ColorPalette>
  <ColorItem
    title='Danger'
    colors={{
      'Reason Danger': palette.danger,
      'Reason Danger Light': palette.dangerLight,
    }}
  />
</ColorPalette>

# ETC.

배경 그라데이션 컬러로 사용되는 기타 컬러입니다.

해당 팔레트는 컴포넌트에서 사용할 수 없습니다.

컴포넌트에서 사용할 컬러를 추가해야 한다면 별도의 팔레트를 생성하세요.

<ColorPalette>
  <ColorItem
    title='theme.color.positive'
    colors={{
      'Reason Serenity 50 ': palette.serenity50,
      'Reason Serenity 100': palette.serenity100,
      'Reason Gray Purple 50': palette.grayPurple50,
    }}
  />
</ColorPalette>
